GATE 2021 contains questions of THREE different types in all the papers:
  1. MCQs (Multiple Choice Questions):
    - Carries 1 or 2 marks each.
    - Are objective in nature, and each will have choice of four answers, out of which ONLY ONE choice is correct.
    - There will be negative marking for a wrong answer chosen in a MCQ, For 1-mark MCQ, 1/3 mark and for 2-mark MCQ, 2/3 mark will be deducted for a wrong answer.
  2. MSQs (Multiple Select Questions):
    - Carries 1 or 2 marks each.
    - Are objective in nature, and each will have choice of four answers, out of which ONE or MORE than ONE choice(s) is / are correct.
    - There will be negative marking for a wrong answer chosen in a MCQ, For 1-mark MCQ, 1/3 mark and for 2-mark MCQ, 2/3 mark will be deducted for a wrong answer.
  3. NATs (Numerical Answer Type):
    - Carries 1 or 2 marks each.
    - For these questions, the answer is a signed real number, which needs to be entered by the candidate using the virtual numeric keypad on the monitor (keyboard of the computer will be disabled). No choices will be shown for these types of questions.
    - The answer may be in decimals as well, for example, 10.1 (one decimal) or 10.01 (two decimals) or -10.001 (three decimals). These questions will be mentioned with, up to which decimal places, the candidates need to present the answer. Also, for some NAT type problems an appropriate range will be considered while evaluating these questions so that the candidate is not unduly penalized due to the usual round-off errors.
    - There is NO Negative Marking
